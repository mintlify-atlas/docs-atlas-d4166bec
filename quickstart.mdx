## Get up and running

This guide walks you through creating a Sorare account, connecting SoGaffer, and submitting your first optimized lineup.

<Steps>
  <Step title="Create a Sorare account">
    If you already have a Sorare account, skip to the next step.

    1. Go to [sorare.com](https://sorare.com) and click **Sign up**.
    2. Enter your email address and choose a password.
    3. Verify your email and complete your profile.

    <Note>
      Sorare requires identity verification before you can buy or trade cards. You can still explore the platform and receive free cards while verification is pending.
    </Note>
  </Step>

  <Step title="Set up SoGaffer">
    Clone the SoGaffer repository and install its dependencies.

    ```bash
    git clone https://github.com/iglesiash/sogaffer.git
    cd sogaffer
    npm install
    ```

    <Tip>
      SoGaffer requires Node.js 18 or later. Run `node --version` to confirm your version before installing.
    </Tip>
  </Step>

  <Step title="Connect SoGaffer to your Sorare account">
    SoGaffer reads your card collection and gameweek data through the Sorare GraphQL API.

    1. Copy the example environment file:

       ```bash
       cp .env.example .env
       ```

    2. Open `.env` in your editor and add your Sorare credentials:

       ```bash .env
       SORARE_EMAIL=you@example.com
       SORARE_PASSWORD=your_password
       ```

    3. Save the file. SoGaffer uses these credentials to fetch your card collection and upcoming fixtures at runtime — they are never stored or transmitted elsewhere.

    <Warning>
      Never commit your `.env` file to version control. The repository's `.gitignore` excludes it by default.
    </Warning>
  </Step>

  <Step title="Fetch your card collection and gameweek data">
    Run the sync command to pull your current card collection and the upcoming gameweek fixtures:

    ```bash
    npm run sync
    ```

    SoGaffer fetches:
    - All cards in your Sorare collection
    - Upcoming fixtures for the current gameweek
    - Recent player performance scores

    The sync typically takes 10–30 seconds depending on the size of your collection.
  </Step>

  <Step title="Generate your optimized lineup">
    Run the optimizer to generate the best possible lineup for the upcoming gameweek:

    ```bash
    npm run optimize
    ```

    SoGaffer outputs a ranked lineup recommendation to your terminal, showing each suggested player, their projected score, and the reasoning behind their selection.

    ```
    Optimized lineup for Gameweek 42
    ─────────────────────────────────
    GK  M. Maignan       proj. 55.2
    CB  V. van Dijk      proj. 61.8
    CB  W. Saliba        proj. 58.4
    CM  M. Fernandes     proj. 67.1
    FW  E. Haaland       proj. 82.3
    ─────────────────────────────────
    Total projected score: 324.8
    ```

    <Tip>
      Run `npm run optimize -- --league all-star` to target a specific competition. See the [building your lineup guide](/guides/building-your-lineup) for all available options.
    </Tip>
  </Step>

  <Step title="Submit your lineup on Sorare">
    Once you have your recommendation, submit the lineup directly on Sorare:

    1. Log in to [sorare.com](https://sorare.com) and open the **Play** tab.
    2. Select the competition you are entering.
    3. Click **+ Add player** for each position and choose the cards SoGaffer recommended.
    4. Review your lineup, then click **Submit**.

    <Note>
      Lineups must be submitted before the gameweek lock time shown on the competition page. SoGaffer displays this deadline in the optimizer output.
    </Note>
  </Step>
</Steps>
